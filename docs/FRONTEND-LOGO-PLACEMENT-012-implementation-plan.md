# Implementation Plan

## Amaç

Yeni SVG logoyu navbar ve ajan çıktılarında daha görünür, sakin animasyonlu ve metinle hizalı göstermek.

## Uygulama

- SVG içindeki geniş şeffaf boşluğu bileşen seviyesinde optik ölçekle telafi et.
- Logo hareketine düşük genlikli süzülme, dönüş ve parıltı ekle.
- Sidebar ve sohbet üst barındaki logo ölçülerini metin bloklarıyla eşleştir.
- Ajan yanıtındaki yuvarlak çerçeveyi kaldır.
- Ajan logosunu büyütüp çıktı metninin üst çizgisine hizala.
- Düşünme durumundaki yuvarlak çerçeveyi de kaldır.
