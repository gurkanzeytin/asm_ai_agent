# Engineering Review

## Değerlendirme

- Biçimlendirme merkezi frontend sunum katmanında tutulur.
- API rotalarına veya sorgu üretimine iş mantığı eklenmedi.
- Bilinmeyen string/dimension alanları metrik kartına dönüştürülmez.
- Bilinen null metrikler açık boş durum bilgisi taşır.

## Risk

Yeni süre veya adet metrikleri kataloğa eklendiğinde doğru kullanıcı biriminin `METRIC_UNITS_TR` sözlüğüne eklenmesi gerekir.
