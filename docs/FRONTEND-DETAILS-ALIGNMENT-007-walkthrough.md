# Frontend Details Alignment 007 - Walkthrough

## Notification

The response-ready toast no longer exposes Sonner's external close button. Its surface uses a near-background color, a restrained border and shadow, and a 12px corner radius while retaining the slim cyan status rail.

## Details Panel

The SQL section header now includes a dedicated copy icon. It copies the exact generated SQL without expanding or collapsing the section and briefly changes to a confirmation icon. The details panel closes with a right arrow; the chat header uses right and left arrows to close or reopen it. The unrelated trash action was removed from the header.

## Alignment

The SQL header checkbox previously used right alignment while row checkboxes used left alignment. Both now share the same width, padding, and left alignment. Assistant Markdown headings retain normal spacing between sections, but the first heading removes its top margin so it begins level with the avatar.
