# Frontend Chat Composer 006 - Walkthrough

## Assistant Messages

Assistant markdown now renders directly on the conversation background. The avatar remains as the response anchor, while the user message retains its primary-colored bubble. The thinking label also renders without a glass card.

## Composer

The separate control row and `Gönder / Yeni satır` shortcut legend were removed. The textarea reserves space on its right edge for a 40px send or stop control. The button is positioned inside the composer, responds to hover and press with Motion, and gives the send icon a restrained idle movement only while input is available.

## Header Typography

`Med Agent` and the current conversation title now use 15px semibold text. `Sağlık Zekâsı` and connection status use 12px text with explicit line heights, improving hierarchy without increasing the header footprint excessively.
