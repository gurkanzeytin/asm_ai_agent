# Implementation Plan

## Amaç

Sol menü, sohbet ve Detaylar panelindeki üst sınır çizgilerini tek yatay eksende hizalamak.

## Uygulama

- Üç üst bar için sabit `h-16` yüksekliği kullan.
- Sol menü başlığına diğer panellerle aynı alt ayıracı ekle.
- Sol menü daraltma düğmesini üst barın dikey merkezine taşı.
- Frontend test, tip kontrolü ve üretim derlemesiyle değişikliği doğrula.
